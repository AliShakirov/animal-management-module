# Модуль учёта животных для фермерского хозяйства

## Описание проекта
Этот проект предназначен для фермеров и владельцев фермерских хозяйств, которым необходимо вести учёт животных. Модуль позволяет добавлять, редактировать и удалять информацию о животных, отслеживать их состояние, а также предоставлять статистику по животным на ферме.

### Основные функции:
- Добавление животных в систему
- Редактирование информации о животных
- Удаление животных
- Просмотр статистики по животным
- Поддержка различных типов животных (коровы, овцы, свиньи и т.д.)
